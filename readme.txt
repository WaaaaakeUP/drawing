Description:
It's a drawing program written in Java.

Controls:
- the buttons on the left are used to change color 
- you can use the slider under the color selection to adjust stroke thickness
- the slider at the bottom supports scrubbing 
- redo/undo button can redraw/clear one stroke
- backward and forward buttons are used to play drawing animation


Enhancements:
- when the canvas resizes to fit the available space, it maintains its aspect ratio 
- click on forward button and backward button will play the animation
- each stoke will need exactly 1 sec to be drawn in the animation.

Development environment:
osx 10.13.4, Java 10.0.1 w.IntelliJ IDEA 2018.1.2
